# Office-Subscription-Analysis
공객 행동 데이터를 활용한 공유오피스 유료 전환 예측 및 K-means 군집 기반 타겟 마케팅
# 🏢 머물고 싶은 사람은 티가 난다: 공유오피스 유료 전환 핵심 동인 분석

> **"무료체험 고객 62.1%가 왜 결제하지 않을까?"**
> 무료 체험 유저의 3일간 행동 데이터를 분석하여 결제 전환 가능성을 예측하고, K-means 군집 분석을 통해 고객별 맞춤형 전략을 제안했습니다.

## 📌 Project Overview
- **배경**: 무료 체험 이후 결제로 이어지지 않는 클래스 불균형(1:1.6) 문제 해결 필요
- **목적**: 유료 전환을 결정짓는 핵심 행동 신호 발굴 및 잠재 고객 발굴 모델 구축
- **나의 역할**: **프로젝트 팀장(Leader)**, 데이터 품질 관리 및 전처리, K-means를 활용한 고객 세그먼테이션 수행

## 🛠️ Tech Stack
- **Machine Learning:** Random Forest (전환 예측), K-means Clustering (고객 분류)
- **Data Engineering:** SMOTE (데이터 불균형 해소), Scaling/Encoding
- **Analysis:** Python (Pandas, Scikit-learn, Seaborn)

## 💡 Key Insights
1. **Day 1의 법칙**: '신청일부터 첫 방문까지의 기간'과 **'첫날(Day 1) 체류 시간'**이 결제 전환율과 가장 강력한 상관관계를 보임.
2. **체험 완주의 효과**: 3일 체험을 모두 완료한 고객의 전환율이 압도적으로 높으며, 재방문 유도가 전체 매출 향상의 핵심임을 입증.
3. **군집별 맞춤 전략**: K-means로 유저를 분류하여 '초기 이탈군'에게는 온보딩 혜택을, '고관여군'에게는 리텐션 강화를 제안.
